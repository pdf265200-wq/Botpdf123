# بوت تحرير PDF - تيليجرام

بوت تيليجرام احترافي لتحرير ملفات PDF مع واجهة عربية كاملة.

## المميزات

- 📑 دمج عدة ملفات PDF
- ✂️ تقسيم ملف PDF
- 🗑️ حذف صفحات محددة
- 📄 استخراج صفحات معينة
- 🔄 تدوير الصفحات
- 🔀 إعادة ترتيب الصفحات
- 📦 ضغط حجم ملف PDF
- 🖼️ تحويل الصور إلى PDF
- 📸 تحويل PDF إلى صور
- 💧 إضافة علامة مائية
- 🔒 إضافة كلمة مرور
- 🔓 إزالة كلمة المرور
- 📝 استخراج النصوص
- ℹ️ عرض معلومات الملف

### المتطلبات

- Python 3.11+
- Token بوت تيليجرام من @BotFather
